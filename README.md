# Calculadora de Factorial

Ejercicio de Lógica de Programación 3 — Generation México

## Descripción

Programa que calcula el factorial de un número ingresado por el usuario. Valida que el dato sea un número entero no negativo y muestra el resultado en pantalla.

## ¿Qué es un factorial?

El factorial de un número es el producto de todos los enteros positivos desde 1 hasta ese número.

Ejemplo:
5! = 1 × 2 × 3 × 4 × 5 = 120

## Cómo usarlo

1. Clona o descarga el repositorio
2. Abre el archivo `index.html` en tu navegador
3. Ingresa un número entero en el campo de texto
4. Haz clic en **Calcular**
5. El resultado aparece en pantalla

## Ejemplos

| Entrada | Salida |
|---------|--------|
| 5       | 120    |
| 6       | 720    |
| 0       | 1      |

## Validaciones

- Si el campo está vacío o el valor no es un número, muestra un mensaje de error
- Si el número es negativo o decimal, muestra un mensaje de error
